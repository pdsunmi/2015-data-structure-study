* 학생 정보 관리 프로그램 

1. db.txt 입력 ( db.txt 파일로 부터 데이터를 불러와 회원정보를 해시테이블로 구축 )

2. 질의 입력

	- 학생 추가 : A S E ( S : 학번 E : 이메일 주소 )
	- 학생 정보 출력 : P S ( S : 학번 )
	- 수강 과목 추가 : U S C ( S : 학번 C : 추가한 수강과목의 학점 )
	- 이메일 주소 변경 : M S E ( S : 학번 E : 이메일 주소 )
	- 프로그램 종료 : Q
